# OnceMutex

> A mutex providing one-time synchronized access, then safe unsynchronized access.

## Usage

Use the crates.io repository; add this to your `Cargo.toml` along
with the rest of your dependencies:

```toml
[dependencies]
once-mutex = "^0.0.0"
```
